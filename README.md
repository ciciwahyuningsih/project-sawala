Nama Project: Website Katalog Produk
OverView Project: Website katalog produk sederhana dibuat untuk menampilkan berbagai pilihan produk secara online serta untuk melakukan pembelian
Fitur: Halaman beranda,kategori,produk,kontak,keranjang, dan pembelian
Teknologi: HTML,CSS,JavaScript
Cara menjalankan project: 1.Download atau clone project dari Github
                          2.Buka folder project
                          3.klik file 'index.html'
                          Website akan terbuka di browser
Struktur folder singkat: *'index.html' (halaman website katalog produk)
                         * Folder 'imagies' (tempat penyimpanan seluruh file foto untuk web)
